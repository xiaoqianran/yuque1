# 本地开发与 M0 基线

## 前置

- Node.js **22**（`.nvmrc`）
- pnpm 9（`packageManager`）
- Docker（Compose：PostgreSQL 16 + Redis 7）

## 一键依赖

```bash
cp .env.example .env
pnpm install
pnpm compose:up
pnpm --filter @yuque1/api exec prisma migrate deploy
pnpm prisma:generate
```

## 启动

```bash
# 终端 1
pnpm dev:api

# 终端 2
pnpm dev:web
```

- Web: http://localhost:5173 （代理 `/api` → API）
- API: http://localhost:3000/api/v1/health

## 质量命令

```bash
pnpm run ci
```

含：Node 检查、docs 检查、prisma generate、typecheck、unit test、build。

## M0 范围

| 有 | 无（后续里程碑） |
|----|------------------|
| monorepo apps/web + apps/api | 业务 Auth/KB/Doc 实现 |
| Prisma schema + init migration | 完整 Session/Redis 登录 |
| Compose PG/Redis | MinIO |
| Health 包络接口 | 编辑器 |

对齐契约：`docs/api/openapi.yaml` 0.1.1。
